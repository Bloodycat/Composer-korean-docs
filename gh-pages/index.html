Composer 한국어 매뉴얼

Composer를 설정하고 이용하는 방법에 대한 한국어 매뉴얼을 제공합니다.

Composer?

composer 는 php 를 개발하는데 있어서 패키지간의 의존성을 관리하는 의존성 관리자입니다. node.js 의 npm, ruby의 bundle과 같은 역할을 수행합니다.

php를 개발하는데 있어서 다양한 패키지들을 설치하고 의존성을 관리하며 Autoload를 사용하는데 Composer가 꼭 필요하다고 할 수 있습니다.

PHP 5.3.2+ 이상이면 Composer가 구동될 수 있습니다.

패키지의 검색

composer 를 통해서 설치할 수 있는 수많은 패키지들을 손쉽게 검색하는 방법은 packagist를 이용하는 방법입니다. 이곳에서 키워드나 패키지명, 벤더의 이름을 기반으로 검색하여 원하는 패키지를 손쉽게 찾을 수 있습니다.

한국어 문서의 번역

Composer의 한글 매뉴얼은 많은 분들의 공헌으로 진행되었습니다. 공헌자 확인하기 오타나 수정사항에 대해서는 @findstar 에게 연락하시거나 github 에서 Pull Request 를 보내주시면 검토후 반영됩니다.
